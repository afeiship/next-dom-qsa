#next-dom-qsa

##description:
+ A modern dom selector based on next toolkit.
+ Simple selector for learning DOM Selector API.
+ user query-selector api,not support lower version browser.


##usage:
+ ID selector:
```javascript
  nx.$('#id1');
```

+ Class selector:
```javascript
  nx.$('.cls2');
```

+ Tag selector:
```javascript
  nx.$('div');
```
+ All selector:
```javascript
  nx.$('div.cls2.cls2[role=dialog]');
```
